ChocAn

Chocoholics Anonymous (ChocAn) is a data processing software that aims to aid in the delivery and recording of healthcare services for individuals battling chocolate addiction. This software provides a comprehensible interface for healthcare providers to record healthcare services administered to those registered with the ChocAn network.
